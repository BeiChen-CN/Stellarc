# Changelog

## 1.0.2 - 2026-02-16

- 新增沉浸模式：随机点名页可仅保留点名按钮、人数选项、退出按钮和抽取结果。
- 增强新手引导：支持直接创建班级、从文件导入学生名单，并可在引导内设置主题/配色/设计风格/抽取动画。
- 优化点名动画与音效：修复候选重复与节奏抖动，新增动画时长滑杆与点名音效强度（低/中/高）。
- 重构设置页面信息架构：拆分外观与体验、迁移模块开关与快捷键位置、危险操作折叠展示。
- 发布流程增强：Release 仅上传安装包并附带 SHA256 校验文件。

## 1.0.1 - 2026-02-15

- 新增规则模板中心，支持模板导入/导出、应用规则、应用到班级任务。
- 新增课堂报告增强导出，支持结构化 CSV 与 Markdown 报告扩展统计。
- 新增积分规则引擎（封顶封底、单次变更限制、同任务当日去重）。
- 新增积分日志面板，支持筛选与单条回滚。
- 优化历史记录：事件筛选、回放增强、可展开详情、长内容显示稳定。
- 优化分组展示：布局可读性提升，支持按组设置任务与分值。
